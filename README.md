# Python Weather Application

This project is a simple Weather Application built using Python.
It fetches real-time weather data using the OpenWeather API and displays it through a graphical user interface.

## Features

* Fetch real-time weather information
* GUI interface built using Tkinter
* Displays temperature, humidity, and weather condition
* Handles invalid city inputs

## Technologies Used

* Python
* Tkinter
* Requests Library
* OpenWeather API

## How to Run

1. Install Python.
2. Install the requests library.
3. Get an API key from OpenWeather.
4. Replace `YOUR_API_KEY` in the code with your API key.
5. Run the program.

## File

* `weather_gui.py` – Main Python GUI weather application
